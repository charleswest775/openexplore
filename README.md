# openexplore

Tools to explore and analyze iPhone backup datasets.

## Purpose

This project provides utilities for reading, parsing, and inspecting the contents of iPhone backups. It is designed to support development efforts where you need to:

- Understand the structure and contents of iPhone backup files
- Validate and cross-reference results from other programs that read iPhone backup data
- Explore SQLite databases, plists, and other artifacts within backups
